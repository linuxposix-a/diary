## Bash

- Utilise GNU Readline qui utilise des raccourcis Emacs
  - `CTRL+A` Curseur au début
  - `CTRL+E` Curseur à la fin
  - `CTRL+L` Efface l'écran
  - `CTRL+C` Envoie SIGINT
  - `CTRL+D` Envoie EOF
